[Catalyst 3500Series의 P/W Recovery방법]
 
1 PC의 하이퍼터미널을 아래설정으로 셋팅한다:
 
9600 baud rate
No parity
8 data bits
1 stop bit
No flow control   전원을 넣는다.
 
  1. 모드버튼을 누른 상태에서 전원을 넣고, 1번 포트가 점멸이 끝날 때 까지 모드버튼을 누른다.
  2. Type flash_init. ß 이 명령이 있어야 Dir Flash:명령으로 플레쉬 메모리가 보인다.
  3. Type load_helper.
  4. Type dir flash:.
    !--- Make sure to type a colon ":" after the dir flash
The switch file system is displayed:

  1. 스위치에 많은 설정이 없을 경우엔 Delete Config.text로 초기화한 다음 다시 설정하면 편리할 때가 많다
 
아래 설명부터는 config.text에 설정을 보존하면서 복구 하는 방법이다.
 
  1. rename flash:config.text flash:config.old 명령으로 원래 설정 파일명을 바꿔준다.
  2. Type boot 로 시스템 재부팅
  3. Enter N at the prompt to start the Setup program, Continue with the configuration dialog? [yes/no] : N
  4.  en 명령으로 enable mode로 진입.
  5. Type rename flash:config.old flash:config.text 명령으로 원래 설정 파일명을 바꿔준다.
  6. 기존의 설정을 메모리로 올린다:
 
Switch# copy flash:config.text system:running-config
Source filename [config.text]? (press Return)
Destination filename [running-config]? (press Return)    
 
  1. 패스워드를 바꿔준다.:
switch#configure terminal
switch(config)#no enable secret
!-- This step is necessary if the switch had an enable secret password
switch(config)#enable password Cisco
switch#(config)#^Z
!--Control/Z
 
  1. 새로운 시스템 설정을 저장한다.:
switch#write memory
